EDUCATION CHATBOT
Project Overview:
Education Chatbot is an AI-powered chatbot developed to help students get instant answers to educational questions. The chatbot can respond to user queries related to subjects, programming, general knowledge, and basic learning topics through an interactive chat interface.

This project combines:
Python backend
HTML, CSS, and JavaScript frontend
AI API integration for intelligent responses

The main purpose of this project is to create a simple virtual learning assistant that improves student learning experience through conversational interaction.

Features:
.Interactive chatbot interface
.Real-time question and answer system
.AI-generated educational responses
.Clean and responsive UI
.Frontend and backend integration
.Easy to use for students
.Beginner-friendly project structure
